A checklist webapp for tracking all tasks and achievements in Clair Obscur Expedition 33
I've used the great excel file built by ErikLeb and Blueye95 as source.
The webapp features:
- checklists for all categories (livros, weapons...)
- primary information for each achievement (location, rewards etc)
- marks achievements that are missable (if you advance past a point in the game)
- marks achievements that require a certain phase (like Act III)
- cross reference achievements and I dos. Example: if you already collected a picto listed as a reward from a boss, the picto carries a checkmark in the boss' rewards info)
- checked achievements are saved locally in Json so they live in your device. I added a button to download the Jason if you want to backup or save advancements to another device 
- connects to respective IGN walkthrough pages for help in executing tasks in case you need
